
# Network Tools Kit

A curated collection of network diagnostic utilities, routing management tools, and secure communication clients for developers and system administrators.

## Contents

| Package | Version | Platform |
|---------|---------|----------|
| NetDebug Client | 7.20.4 | Windows, macOS, Linux |
| Route Manager | 2.4.7 | Windows, macOS, Linux |
| Secure Tunnel | 1.13.11 | Windows, Linux |
| NetKit Terminal | 4.0.1 | Windows, macOS, Linux |
| NetDebug Mobile | 7.20.4 | Android |
| ProxyBox | 1.3.8 | Android |

## Downloads

All binaries are available in [Releases](https://github.com/saramatos534-crypto/network-tools-kit/releases).

## Quick Start

1. Download the appropriate package for your platform
2. Extract the archive
3. Run the executable
